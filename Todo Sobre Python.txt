El Algoritmo:

Una serie de pasos para resulver un problema,
Ejemplo: hacer ejercicio, si quieres estar saludable hay que tener una buena forma ¿como consiguirla? haciendo rutinas, actividades 
como levantar pesas entretar, comer alimentarnos sanamente, y descarsar el dia. todo eso lo vas a repetir otra vez cada dia ¿como termino?
hazta que te quedes satisfecho con los resultados que hayas alcansado algo real en tu salud. eso es un algoritmo una serie de pasos para resulver
un problema ¿pero cual fue el problema? mi salud, mi peso tener un cuerpo mas saludable.
El Algoritmo tiene caracteristicas especiales, que es finito dede tener un incio y un final. tener un incio claro y un final claro, una serie
de pasos que nos lleve ese principio a ese final. siempre un Algoritmo es Finito no tiene pasos definidos siempre tenemos una cantidad de limitado
de pasos para poder resulver ese problema,
Un algoritmo No Es Ambiguo. no podemos tener pasos que se signifique algo en un contexto y que en otro contexto se signifique otra cosa, en le algoritmo
necesitamos tener cada paso con una finalidad en particular, tiene que hacer una cosa de manera unirica, "que uno se signifique algo y en otro otra cosa"

Palabras reservadas en Python
Cómo menciamos en vídes anteriores, existen ciertas palabras las cuales no podemos utilizar para nombrar a nuestras variables, funciones o clases. A este tipo de palabras las conoceremos como palabras reservadas. Y estan diseñadas para realizar tareas especificar por parte del lenguajes.

Aquí el listado de todas ellas:

False	class	is	return
None	continue	lambda	try
True	def	nonlocal	while
and	del	not	with
as	elif	or	yield
assert	else	pass	*
break	except	raise	*
